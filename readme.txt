== Description ==

Scheduled Post Reminder Notifications is a simple yet powerful plugin that helps you stay on top of your scheduled posts. It sends reminders via email or dashboard notifications before your posts are published, ensuring that you never miss an important content update.

Features

- Customizable Reminder Timing: Set how long before the scheduled time you want to receive a reminder (in minutes).
- Flexible Notification Methods: Choose between email notifications or WordPress dashboard notifications.
- User-friendly Settings: Easily configure your preferences from a simple settings page.

Use Cases

- Bloggers and Content Creators: Get notified in advance to prepare promotional activities for your upcoming posts.
- Content Teams: Ensure all team members are aware of the publishing schedule and ready for action.
- Businesses: Maintain a consistent content schedule by getting timely reminders for all your scheduled posts.
- Freelancers and Agencies: Improve client satisfaction by reminding clients of upcoming content for approval.

== Installation ==

1. Upload the `scheduled-post-reminder-notifications` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Settings' -> 'Scheduled Post Reminder Notifications' to configure the plugin.

== Frequently Asked Questions ==

= How do I set the reminder time? =
Go to the plugin's settings page under 'Settings' -> 'Scheduled Post Reminder Notifications' and set the reminder time in minutes.

= Can I choose how I receive notifications? =
Yes, you can choose between email notifications and dashboard notifications from the settings page.

= What happens if I deactivate the plugin? =
If you deactivate the plugin, all scheduled reminder events will be cleared to prevent unnecessary notifications.
